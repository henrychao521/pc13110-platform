MODULE 3-1・從手算到電腦模擬

電腦輔助結構分析 FEA

複雜的結構,手算幾乎不可能。工程師改用電腦把物體切成成千上萬個小元素,逐一計算——這就是有限元素分析。

什麼是有限元素分析(FEA)?

有限元素分析(Finite Element Analysis,FEA)是電腦輔助工程(CAE)的核心技術。 它把一個複雜的物體,在電腦裡切割成許多微小、形狀規則的「元素(element)」—— 這個過程稱為「網格劃分(meshing)」。

接著,電腦對每一個小元素套用力學方程式,再把成千上萬個元素的結果組合起來, 就能算出整個結構在受力下的應力分布、變形量。手算辦不到的複雜結構, 電腦能在幾分鐘內完成。

FEA 模擬的操作步驟

以一個 L 型支架為例,用 Fusion 360、Onshape 等軟體進行結構分析,流程如下:

① 模型準備
繪製 3D 模型,並進行「模型淨化(Defeaturing)」——移除不影響結構分析的微小特徵(如裝飾性圓角、文字雕刻),簡化計算。
② 進入模擬環境
切換到模擬工作空間,選擇分析類型(如「靜態應力」)。
③ 材料定義
為模型指定材料,軟體會載入該材料的楊氏模數、蒲松比、降伏強度等性質。
④ 施加約束
模擬安裝方式,把固定的面設為「固定(Fixed)」約束。
⑤ 施加負載
加上結構實際會承受的力。
⑥ 求解與判讀
執行運算,觀察應力雲圖、位移分布與安全係數。
🧪 動手操作:L 型支架應力模擬器

找出結構最脆弱的地方

L 型支架的垂直面用螺絲固定在牆上,水平端承受向下載重。調整材料、厚度與載重, 觀察應力雲圖——顏色越紅代表應力越大、越接近破壞。

💡 知識小百科

馮·米塞斯應力與安全係數

馮·米塞斯應力(Von Mises stress):把複雜的多方向應力,換算成一個能直接 與材料強度比較的「等效應力」。FEA 結果的彩色雲圖,通常就是它——紅色代表應力最高、最易破壞之處。

安全係數(Safety Factor)= 材料降伏強度 ÷ 最大應力。 安全係數大於 1 才不會破壞;工程上通常要求 2 以上,預留安全餘裕。L 型支架的應力會 集中在「內側轉角」——這正是設計時要特別補強的地方。

🔍 觀念釐清:為什麼要「模型淨化」?
  • 真實 3D 模型常有圓角、刻字、小孔等細節,對整體結構強度幾乎沒影響。
  • 但這些細節會讓網格切得非常細、元素暴增,拖慢甚至卡住運算。
  • 「模型淨化」先移除這些不重要的特徵,讓分析又快又穩——這是 FEA 的第一個重要技巧。
🖥 上機實作:跟著做一次 FEA

用開源 FreeCAD 完成 L 型支架的有限元素分析

上方學了 FEA 的觀念與操作步驟,這裡帶你用完全免費、開源的工具實際走過一次。 FreeCAD 是業界與教育界廣泛使用的開源 CAD/CAE 軟體(LGPL 授權,Windows/macOS/Linux 皆可),內建 FEM Workbench,呼叫 CalculiX(GPL 求解器)執行 FEA 計算。 教室或家用電腦皆可安裝,沒有試用期。

🧩 FreeCAD ©2002–present, The FreeCAD Project,LGPL 2.1+;CalculiX ©Guido Dhondt,GPL v2。兩者皆為完全免費的開源軟體。
① 下載與安裝
到 freecad.org 下載對應作業系統的安裝檔(約 500 MB)。安裝完成後啟動,左上 Workbench 切換到 FEM。
② 建立或匯入模型
用 Part Design 畫一個簡單 L 型支架(或匯入課本提供的 STEP 檔)。先移除不影響結構強度的圓角、刻字、小孔——這就是「模型淨化」。
③ 建立 FEM 分析容器
點工具列「Analysis container」,系統會自動帶入 Solver(CalculiXccxTools)、Material、Constraints 等預設步驟。材料先選「Steel」(或課堂指定的鋼/鋁)。
④ 設定邊界條件
在支架要固定的安裝面加入 Constraint fixed(完全約束);在受力面加入 Constraint force,輸入力的大小與方向(例如 100 N,沿 -Z)。
⑤ 網格切割
右鍵 Analysis container → 加入 Mesh / Gmsh,讓 FreeCAD 把物體切成有限元素。先用預設參數跑一次;結果若不夠細緻再縮小元素尺寸。
⑥ 執行求解
雙擊 Solver,點 Write .inp file 產出輸入檔,再點 Run CalculiX 啟動求解(視模型大小,幾秒到幾分鐘)。
⑦ 檢視結果雲圖
求解完成後,雙擊結果項目並選 von Mises Stress,即可看到應力雲圖。紅色為應力最大、最易破壞處——驗證:它是不是出現在「內側轉角」?與課堂講解一致。
