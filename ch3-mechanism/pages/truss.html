MODULE 3-1・桁架結構

桁架結構解算

橋梁、屋架、起重機骨架,背後幾乎都是「桁架」。它用最少的材料,撐起最大的負載——關鍵就在三角形。

桁架是什麼?

桁架(Truss)是由一根根桿件(member),在端點以 節點(joint)連接而成的結構。它的核心是三角形——三角形是 唯一無法被「壓變形」的多邊形,所以桁架特別穩固。

桁架受力時,每根桿件只承受沿桿件方向的力,不是被「拉長」(張力)、 就是被「壓短」(壓力)。分析桁架,就是要算出每根桿件到底承受多大的張力或壓力。

🧪 動手操作:桁架受力視覺化

三種桁架,受力大不同

切換三種經典桁架,調整中央載重,觀察每根桿件的受力狀態—— 紅色=張力(被拉)、 藍色=壓力(被壓),線條越粗代表受力越大。

💡 知識小百科

三種經典桁架的差異

Pratt(普拉特式):斜桿朝中央向下傾斜。受力時斜桿為張力、直桿為壓力。 因為較長的斜桿承受張力(鋼材抗拉性佳),最省鋼材,是鋼橋最常見的形式。
Howe(豪威式):斜桿方向與 Pratt 相反。受力時斜桿為壓力、直桿為張力, 較適合木造桁架(木材抗壓性佳)。
Warren(華倫式):斜桿呈 W 形交錯,沒有直桿(或只有少數)。斜桿 張力、壓力交替出現,結構簡潔、桿件數少。

這個桁架是靜定還是靜不定?

用公式 n = b + r − 2j 判斷:b 桿件數、r 反力數、j 節點數。 輸入數值,看計算結果。

深入探究:n < 0、n = 0、n > 0 各代表什麼?

n < 0(不穩定):桿件不夠,結構會垮掉——這不是合格的結構。
n = 0(靜定):桿件剛剛好,可用基礎平衡方程式完全解出每根桿件的力。
n > 0(靜不定):桿件「多」了,光靠平衡方程式解不出來,需要再考慮材料變形 ——這時就要靠電腦的 FEA 分析。多出來的桿件雖然增加分析難度,卻也讓結構更穩固、更安全。
