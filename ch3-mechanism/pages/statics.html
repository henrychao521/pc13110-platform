靜力學與自由體圖

第 2 章我們確認設計「做得出來」;本章要確認它「撐得住」。一切結構分析,都從一張「自由體圖」開始。

自由體圖（Free Body Diagram, FBD)

繪製自由體圖是結構分析中第一步、也是最重要的一步。它的目的,是把要分析的物體 （或其中一部分)從周圍環境中「分離」出來,並畫出施加在物體上的所有外力:

・主動力:像負載、重力等主動施加的力。
・反作用力:支撐點與連接處對物體產生的支撐力。

把所有力清楚標示出來後,才能進一步用數學求解。

平衡方程式(Equilibrium Equations)

📐 核心原理

若物體處於靜止平衡狀態,則其所受淨力與淨力矩均為零。

把 3D 模型轉換為平面 2D 來分析時,可使用以下三條平衡方程式:
・水平方向淨力為零:ΣFx = 0
・垂直方向淨力為零:ΣFy = 0
・對任一點的淨力矩為零:ΣM = 0
透過這些方程式,就能求出未知的支撐反力或內力。

深入探究:為什麼載重越靠近 A,RA 就越大?

對 A 點取力矩平衡:RB × L = P × a(a 是載重距 A 的距離,L 是梁長)。 所以 RB = P·a / L,載重越靠近 A(a 越小),RB 就越小; 再由 ΣFy=0 得 RA = P − RB,所以 RA 就越大。

這就是「翹翹板原理」——力矩 = 力 × 力臂。離支點越遠的力,造成的轉動效果越強。

💡 知識小百科

靜不定結構

一個系統的靜不定度,表示系統中「未知量」比「靜力方程式」還多的數量。 以桁架(Truss)結構來說,計算公式為 n = b + r − 2j: b 為桿件數、r 為反力數、j 為節點數。

若 n = 0 為靜定結構;n > 0 為靜不定結構,超過的數量即靜不定度。 靜不定度越大,系統的「穩定性」通常越強——下一個模組會深入練習。

手算分析的局限

上述方法適用於靜止、固定且形狀簡單的結構。但對於更複雜的結構 (如不規則的桁架、連續梁)或超靜定結構,手動計算會變得非常繁瑣,甚至無法完成。 這時,電腦模擬分析就顯得格外重要——這正是後面 FEA 模組的主題。
