# PC13110 工程設計學習平台 — 安裝與移植說明

> 本文件說明如何把本平台從 GitHub Pages 移植到其他主機 / 平台。
> 對應 GitHub repo：https://github.com/henrychao521/pc13110-platform
> 封存版本 commit：`af9dbef`（封存日期：2026-06-04）

---

## 一、平台特性（移植前先了解）

| 項目 | 說明 |
|---|---|
| **技術架構** | 純前端靜態網站，HTML5 + CSS3 + Vanilla JavaScript |
| **建置工具** | ❌ 無。不需要 npm / webpack / node build，**不需要編譯** |
| **後端** | ❌ 主平台零後端。資料存瀏覽器 `localStorage` |
| **函式庫** | 全部已下載進 `vendor/`（three.js、Leaflet、jsMind、Frappe Gantt、qrcode），無 CDN 相依 |
| **路徑** | ✅ **全站使用相對路徑**，沒有任何綁死 `/pc13110-platform/` 子路徑的絕對連結 → 可放在任何網域的**根目錄或任意子目錄**都能運作 |
| **檔案規模** | 約 423 個檔案、解壓後約 130 MB（主要是 `arcade/`73MB 遊戲與街景照、`assets/`24MB） |

**核心結論：移植 = 把檔案丟上任何能提供靜態檔案的伺服器即可。沒有任何建置步驟。**

## 二、最簡單的移植方式（依平台分類）

### 方式 A：任何靜態網頁主機（最推薦）
適用：Netlify、Vercel、Cloudflare Pages、GitLab Pages、Firebase Hosting、自家 Apache/Nginx、學校網站空間。

1. 解壓本封存檔。
2. 把**整個資料夾內容**（`index.html` 與所有子目錄）上傳到主機的網站根目錄。
3. 確認該主機把 `index.html` 當預設首頁。
4. 完成，直接用瀏覽器開即可。

> Netlify / Cloudflare Pages：直接把資料夾拖進部署頁面即可，無需任何設定。

### 方式 B：本機 / 區域網路測試（給老師上課用）
需要的只是任何一個靜態檔案伺服器，擇一：

```bash
# 解壓後進入專案資料夾，三選一：

# 1) Python（macOS/Linux 內建）
python3 -m http.server 8080

# 2) Node（若已安裝）
npx serve -l 8080

# 3) PHP（若已安裝）
php -S 0.0.0.0:8080
```

瀏覽器開 `http://localhost:8080`（同網段其他裝置用 `http://<本機IP>:8080`）。

> ⚠️ **不要用「直接雙擊 HTML 用 file:// 開啟」**。部分模組以 ES module（`import`）載入 three.js，`file://` 會被瀏覽器 CORS 擋下。**一定要透過 http 伺服器開啟。**

### 方式 C：另一個 GitHub Pages
1. 新建 repo → 上傳全部檔案 → Settings → Pages → 來源選分支 root。
2. 因全為相對路徑，換 repo 名稱也不需改任何程式碼。

## 三、選用功能：多人連線伺服器（像素實驗室）

`lobby.html`（像素工程實驗室）支援多位學生同時上線看到彼此。這是**唯一需要後端的功能，且為選用**。

- 伺服器檔案：`server/server.js`
- **純 Node.js、零相依套件，不需 `npm install`**
- 啟動：
  ```bash
  node server/server.js          # 預設埠 8732
  PORT=9000 node server/server.js  # 自訂埠
  ```
- 需求：Node.js（建議 v18 以上，本機實測 v25 可用）
- **連不到伺服器時，像素實驗室會自動退回單人模式**，所以靜態託管（如 GitHub Pages / Netlify）不啟此伺服器也能正常使用，只是沒有多人功能。
- 若要正式多人上課：把 `server.js` 跑在一台學生連得到的主機上，並在前端設定對應的 WebSocket 位址（詳見 `server/README.md`）。HTTPS 網站需搭配 wss/反向代理。

## 四、執行時的外部相依（需網路）

平台主體可離線運作，但**以下功能在執行時會連外**。移植後若該功能異常，多半是這些外部服務或金鑰問題，**不是平台本身**：

| 功能 | 外部服務 | 沒有時的行為 |
|---|---|---|
| Google Fonts（46 個頁面） | fonts.googleapis.com | 退回系統字型，不影響功能 |
| 街景挑戰 360° 環景（arcade geoguess） | Mapillary API | **需使用者自行在瀏覽器輸入 token**（存 localStorage `pc13110_mapillary_token`，不在程式碼內）；無 token 自動退回 Esri 衛星空照 |
| 猜測地圖圖磚 | CartoDB / Leaflet 圖磚 | 需連網 |
| 交誼廳科技新聞 | Hacker News API | 失敗退回內建內容 |
| 交誼廳氣象 | Open-Meteo（免金鑰） | 失敗退回台北 |
| 部分章節嵌入式工具 | Wokwi、Falstad CircuitJS、PhET、PMKS+、Tinkercad 等 iframe | 需連網；皆為官方公開服務 |
| 延伸學習參考連結 | 維基百科等（151 筆） | 純超連結，點擊才連 |

> 若要做到「完全離線」教室：Google Fonts 可自行下載字型改本地引用；地圖/街景/嵌入式模擬器因屬第三方服務，無法離線。

- [ ] （選用）多人功能：啟動 `node server/server.js` 後，`lobby.html` 右上連線狀態顯示已連線
- [ ] （選用）街景挑戰：輸入 Mapillary token 後環景可拖曳
