# 工程實驗室 — 多人連線伺服器

讓多位學生在像素實驗室裡同時看到彼此的角色走動。

## 啟動

需要 Node.js（不需安裝任何套件，零相依）：

```bash
node server/server.js
```

啟動後終端機會印出可連線的位址，例如：

```
本機:      ws://localhost:8732
區域網路:  ws://192.168.x.x:8732   (手機連這個)
```

## 連線方式

- **同一台電腦**：用 `http://localhost:8731/lobby.html` 開啟平台即可，網頁會自動連到 `ws://localhost:8732`。
- **手機 / 其他裝置**：與電腦連同一個 Wi-Fi，用電腦的區域網路 IP 開啟平台
  （例如 `http://192.168.x.x:8731/lobby.html`），網頁會自動連到同一台的 WebSocket 伺服器。
- 連不到伺服器時，實驗室會自動以**單人模式**運作，不影響使用。

## 連接埠

預設 `8732`，可用環境變數覆寫：

```bash
PORT=9000 node server/server.js
```

## 備註

- 本伺服器為純前端平台的「第二階段」功能，GitHub Pages 無法執行伺服器，
  故線上版維持單人模式；多人連線目前用於本機 / 區域網路，日後可移植到其他主機。
