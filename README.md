# PC13110 工程設計學習平台

> 對應普通型高中「生活科技」教科書（PC13110，趙珩宇 編）的互動學習平台。

把課本變成可以動手操作的工程實驗室——從科技趨勢、CAD 建模、結構模擬，到電路設計與
ESP32 程式燒錄，每一章都讓學生在瀏覽器內，用真實的開源工具走過一次完整的工程設計流程。

## 目前進度（第一波）

- ✅ **Hub 首頁**：工程設計流程七階段互動環、五章導覽、學習進度
- ✅ **第 1 章 加速發展的科技與工程**
  - 科技趨勢儀表板（AI／量子半導體／生醫／再生能源 互動圖表）
  - 工程設計流程互動（STEM 齒輪、流程七階段、界定問題）
  - 創意思考工具箱（六頂思考帽／SCAMPER／曼陀羅九宮格）
  - 專題規劃器（心智圖 + 甘特圖）
  - 競賽與職涯倫理（競賽圖鑑、工程倫理情境題）
- ⏳ 第 2–5 章開發中

## 技術架構

- 純前端、零建置工具（HTML5 + CSS3 + Vanilla JavaScript）
- 進度儲存於瀏覽器 `localStorage`
- Web Audio API 程式生成音效

## 快速開始

```bash
# 在專案根目錄啟動本機伺服器
python3 -m http.server 8080
# 瀏覽器開啟 http://localhost:8080
```
